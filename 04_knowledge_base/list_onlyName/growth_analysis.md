## 성장 모델

### 현재 KB 완성도 추정

알려진 보안 지식 베이스 크기:
- MITRE ATT&CK: ~200 technique + ~600 sub-technique = ~800
- CWE: ~900 항목
- NIST SP 800-53: ~1,200 controls
- OWASP (전체): ~150 항목
- 인프라 컴포넌트: ~200-300
- 프로토콜/기술: ~200
- 도구: ~200
- 일반 보안 개념: ~300-500

**보안 도메인 전체 우주 추정: ~2,500-4,000**

현재:
- 기존 KB: 560
- 발견된 누락: 550
- **커버리지: ~1,100 / ~3,000 ≈ 37%**
